# 🧱 Tinkercad Basics

Welcome. This lesson is about learning the tools you need to build anything in Tinkercad.

---

## 🎯 Goal

By the end of this lesson, you will be able to:

- Move and resize shapes
- Combine simple objects
- Build a basic 3D structure

---

## 🧰 The Core Tools

Tinkercad only needs a few tools to start:

- Shapes
- Move
- Resize
- Rotate
- Duplicate

You don’t need anything fancy.

---

<details>
<summary><strong>Step 1 — Add a Shape</strong></summary>

<pre>

• Drag a shape onto the workplane  
• Start with a box or cylinder  
• This is your building block  

</pre>

</details>

<details>
<summary><strong>Step 2 — Move It</strong></summary>

<pre>

• Click and drag to move the shape  
• Use the arrows for precision  
• Place it where you want it  

</pre>

</details>

<details>
<summary><strong>Step 3 — Resize</strong></summary>

<pre>

• Click the white squares  
• Pull to make it bigger or smaller  
• Try making a tall and a flat shape  

</pre>

</details>

<details>
<summary><strong>Step 4 — Combine Shapes</strong></summary>

<pre>

• Place shapes next to each other  
• Overlap them  
• This is how real objects are built  

</pre>

</details>

<details>
<summary><strong>Step 5 — Duplicate</strong></summary>

<pre>

• Copy shapes you already made  
• Use CTRL + D (or CMD + D)  
• Build faster, not harder  

</pre>

</details>
